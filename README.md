# Career Guide Assistant

This project is a simple web page that showcases a Watson Assistant chatbot for a career guidance experience.

## About Watson Assistant
Watson Assistant is an AI service by IBM that helps build chatbots and virtual assistants. It can answer questions, guide users, and provide helpful support in a conversational way.

## What this project does
- Shows a simple assistant-based webpage
- Uses Watson Assistant to interact with visitors
- Presents a career guidance concept in a clean and easy-to-use interface
